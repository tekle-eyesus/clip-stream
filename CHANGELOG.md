# Changelog

All notable changes to this project will be documented in this file.

## 0.0.2

- Added clip notes with a maximum of 10 words per item.
- Added reorder mode with drag-and-drop shuffling inside pinned and unpinned groups.
- Fixed deleted latest clipboard items reappearing due to clipboard value normalization mismatch.
- Improved sidebar controls and keyboard hinting for notes and reorder mode.

## 0.0.1

- Initial release of Clip Stream.
- Clipboard capture with persistent history.
- Insert, copy, delete, and clear-all actions.
- Search, pin/unpin, and keyboard navigation.
